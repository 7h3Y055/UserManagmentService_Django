### `/account/delete/user/`
Endpoint for deleting the current user's account.

#### Method: `DELETE`

#### Request:
* **URL:** `/account/delete/user/`

#### Response:
* **Success:**
  * *None*

#### HTTP Codes

| Code | Description                |
|------|----------------------------|
| 204  | Account deleted successfully |
